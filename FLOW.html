3. User lifecycle state machine

stateDiagram-v2
    [*] --> new_user : /start
    new_user --> onboarding : tap ➕ setup_profile
    onboarding : profile fields answered ONE-BY-ONE
    onboarding --> photo_prompt : all 6 fields set
    photo_prompt --> face_verify : photo received
    face_verify --> verified : isMatch=true
    face_verify --> photo_prompt : rejected
    verified --> ready_to_match : show 💕 Start Matching
    ready_to_match --> matching : tap 💕
    matching --> matched : pair found
    matching --> ready_to_match : none found
    matched --> chatting : user texts
    chatting --> chatting : normal msg (texts_used += 1)
    chatting --> paywall : texts_used >= 10
    paywall --> chatting : referral success (-10 used)
    paywall --> premium : PayPal checkout
    premium --> chatting : unlimited again
    note right of chatting : Female users bypass paywall
    note right of paywall : -10 only when referred friend buys premium
